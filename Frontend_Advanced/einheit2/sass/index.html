Installation

Mac

  # Homebrew installieren
  # https://brew.sh/
  /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  brew doctor

  # Sass installieren
  brew install sass/sass/sass
  sass -v

Windows

  # Chocolately installieren
  # https://chocolatey.org/
  choco outdated
  
  # Sass installieren
  choco install sass
  sass -v

Kompilierung

  sass projekt/scss/main.scss:projekt/css/main.css

Kompilierung laufend

  sass --watch projekt/scss:projekt/css

Mit Komprimierung

  sass --watch scss:css --style compressed

Nesting

Einfaches Nesting

  ul {
    li {
      color: red;
    }
  }

Nesting: & = Parent Selector

  a {
    font-weight: bold;
  
    &:link {
      text-decoration: none;
    }
  
    &:hover {
      text-decoration: underline;
    }
  }

Nesting: & = Parent Selector

  .form {
    border: 1px solid red;
  
    &__item {
      background-color: red;
    }
  }

Parent Nesting

  li {
    color: blue;
  
    ul & {
      color: red;
    }
  }

Media Query Nesting

  .element {
    width: 300px;
    
    @media screen and (min-width: 400px) {
      width: 500px;
    }
  }

Mixins

  @mixin wrapper($maxWidth) {
    width: 100%;
    max-width: $maxWidth;
    margin: 0 auto;
  }
  
  .box {
    @include wrapper(960px); 
  }

Extend/Inheritance

  .message {
    border: 1px solid #ccc;
    padding: 10px;
    color: #333;
  }
  
  .success {
    @extend .message;
    border-color: green;
  }
  
  .error {
    @extend .message;
    border-color: red;
  }

Operatoren

Es ist möglich in Sass zu rechnen: +, -, *, /, and %.

  article {
      width: 100% / 4;
  }

if-Anweisungen

  @mixin alert($type) {
    @if $type == success {
      background-color: green;
    } @else if $type == error {
      background-color: red;
    } @else {
      background-color: blue;
    }
  }
  
  .message {
      @include alert('success'); 
  }

for-Loops

  @for $i from 1 through 3 {
    .item-#{$i} { width: 2em * $i; }
  }
